# Flappy Bird Game

This is a simple implementation of the classic game Flappy Bird using HTML, CSS, and JavaScript. It uses the following libraries:
- p5.js
- p5.play.js
- p5.sound.min.js
- p5.dom.min.js

You can play the game at https://andy-the-coder.github.io/flappy-bird/.
